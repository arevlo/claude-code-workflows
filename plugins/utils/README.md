# utils

Standalone developer utilities and productivity tools.

## Installation

```
/plugin install utils@claude-code-workflows
```

## Commands

| Command | Description |
|---------|-------------|
| `/dropbox-list` | Browse and list files in Dropbox via rclone |
| `/mac-nosleep` | Toggle macOS sleep prevention (keep Mac awake with lid closed) |
| `/mcp-toggle` | Interactive manager for enabling/disabling MCP servers |
| `/notes-list` | Browse and read Apple Notes from the CLI |
| `/portless-setup` | Set up portless dev URLs for a project (wraps dev script, creates deploy skill) |
| `/proj-architecture` | Generate an interactive HTML architecture dashboard for the current project |
| `/youtube-transcribe` | Extract YouTube captions and save as structured markdown with summary |
| `/interview-feedback` | Generate structured hiring feedback from interview transcripts |
| `/webmcp-brainstorm` | Brainstorm WebMCP tool ideas from project context (design-only, no code) |
